### 문제
자연수 N과 정수 K가 주어졌을 때 이항 계수 를 10,007로 나눈 나머지를 구하는 프로그램을 작성하시오.

### 입력
첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ K ≤ N)

### 출력
이항 계수를 10,007로 나눈 나머지를 출력한다.


```python
while True:
    a = list(map(int, input().split()))
    if a[0]==0 and a[1]==0:
        break
    if a[0] % a[1] == 0:
        print('multiple')
    elif a[1] % a[0] ==0 :
        print('factor')
    else :
        print('neither')
```

### 코드 설명
factorial 함수를 이용해서 이항 계수를 구한 후 10007로 나눈 값을 출력해준다.
